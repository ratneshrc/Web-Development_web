Experiment 12: XSLT to Transform XML

Name: Ratnesh Chavan
Roll No: 57

XML Source

<students>
  <student id="57">
    <name>Ratnesh Chavan</name>
    <department>Computer Science</department>
  </student>
</students>

XSLT Stylesheet

<xsl:stylesheet version="1.0" xmlns:xsl="http://www.w3.org/1999/XSL/Transform">
  <xsl:template match="/">
    <html>
      <body>
        <h2>Student Details</h2>
        <table border="1">
          <tr><th>Roll No</th><th>Student Name</th><th>Department</th></tr>
          <xsl:for-each select="students/student">
            <tr>
              <td><xsl:value-of select="@id" /></td>
              <td><xsl:value-of select="name" /></td>
              <td><xsl:value-of select="department" /></td>
            </tr>
          </xsl:for-each>
        </table>
      </body>
    </html>
  </xsl:template>
</xsl:stylesheet>

Transformed Output

When the XML is transformed with the XSLT stylesheet, the rendered output becomes an HTML table:

Roll NoStudent NameDepartment
57Ratnesh ChavanComputer Science
